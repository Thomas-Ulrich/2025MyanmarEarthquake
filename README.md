# Dynamic Rupture Modeling of the 2025 Mw7.8 Mandalay (Myanmar) Earthquake

This repository contains the input files and scripts used in our study of the
2025 Mw7.8 Myanmar Earthquake, currently under review:

> Ulrich, T., Zou, X., Marchandon, M., Schliwa, N., Tan, F., Gabriel, A.A., 
  Fan, W., Shearer, P., Thant, M., Tin, T.Z.H., Lindsey, E. & Fialko, Y. (2025). 
  *On predictability of slip, rupture geometry, and rupture speed of the Mw7.8 2025 Mandalay (Myanmar) Earthquake.*
> [https://www.researchsquare.com/article/rs-7553399/v1](https://www.researchsquare.com/article/rs-7553399/v1)

## Contents

- **CAD and mesh generation scripts** for all scripts for processing the geodetic model, 
    assembling a CAD model and generating a mesh.
  *(see the `geometry/` folder)*

- **Figure reproduction scripts** for all figures related with the dynamic rupture simulations
    shown in the manuscript
  *(see the `figures/` folder)*

- **Dynamic rupture workflow** to generate an ensemble of dynamic rupture
 scenarios and identify the preferred model
  *(see the `seissol_setup/` folder and its `README.md` for detailed
 instructions)*

- **Velocity model related script**, used to plot 
 the 1D velocity model used 
  *(see `velocity_model/`)*
